# Example: Pages Router (`pages/`)

A standalone, minimal app using the **older `pages/` directory** — included only
to contrast the `pages` top-level folder with the App Router used in the main
lab.

## Run it

```bash
cd examples/pages-router
npm install
npm run dev      # http://localhost:3000
```

- `/` → `pages/index.tsx`
- `/api/hello` → `pages/api/hello.ts`

See `GUIDE.md` for App Router vs Pages Router differences.
